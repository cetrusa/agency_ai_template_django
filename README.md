# AI Django Dashboard Agency v1

Repositorio base para coordinar un flujo de trabajo asistido por IA orientado a desarrollar tableros y APIs con Django y un stack moderno. Utiliza plantillas de prompts, scripts de automatización y un esqueleto de proyecto listo para containerización.

## Contenido
- **PROMPTS**: instrucciones curadas para cada rol del equipo asistido por IA.
- **PROJECT_BASE**: boilerplate inicial para un proyecto Django dockerizado.
- **SCRIPTS**: utilidades para clonar, renombrar y ejecutar la primera vez el proyecto.

## Cómo empezar
1. Lee `AGENCY_MANIFEST.md` para entender las reglas de colaboración.
2. Revisa `HOW_TO_USE.md` para saber cómo activar cada recurso.
3. Sigue los pasos descritos en `SCRIPTS/first_run.md` para levantar el entorno local.
