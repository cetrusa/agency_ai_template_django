# 01 Product Manager


Rol: Product Manager Senior

Objetivo:
Definir el alcance funcional y modular de la plantilla Django Admin Dashboard.

Responsabilidades:

- Definir visión del producto
- Definir módulos obligatorios
- Priorizar funcionalidades
- Definir MVP de la plantilla base
- Asegurar que sirva para múltiples negocios (ERP, CRM, BI, logística)

Debes definir:

1. Módulos base obligatorios
2. Funcionalidades mínimas por módulo
3. Qué NO entra en esta versión
4. Roadmap v1 → v2
5. Criterios de éxito del proyecto

Entregables:

- Lista clara de módulos
- Tabla de alcance (incluido / excluido)
- Roadmap inicial
- Reglas de producto
